# 정부광고 데이터 대시보드 (2019–2026)

한국언론진흥재단이 공시한 **정부광고 집행내역** 8개 연도(2019–2026)를 한 화면에서 탐색하는 정적 웹 대시보드.

🔗 **[대시보드 바로가기](https://leejeonghwan.github.io/gov-ad-dashboard/)**

## 무엇이 들어 있나

총 **1,542,058건** / **5,800여 발주 기관** / **11,300여 매체** 기준으로 다음을 보여준다.

- **전체 시장 개요** — 연도별 집행 추이, 매체구분(옥외/인터넷/인쇄/방송/해외)·기관분류(중앙/지방/공공기관 등) 비중, Top 30 광고주·매체
- **광고주별 분석** — 기관명으로 검색해 해당 발주처가 어느 매체에 얼마나 집행했는지
- **매체별 분석** — 매체명으로 검색해 어느 기관에서 얼마나 수주했는지
- 연도·매체구분·기관분류 다중 필터

## 데이터 출처

- [한국언론진흥재단](https://www.kpf.or.kr) 정부광고 집행내역 공시
- 원본 XLSX 파일 8개(2019–2026년도)를 직접 읽어 병합·집계
- 행의 연도는 **파일명에서 추출** (광고시작일 컬럼은 입력 오류가 많아 사용하지 않음)
- 본 저장소에는 가공 후 집계 JSON만 포함 (원본 XLSX는 대용량이라 제외)

## 기술 스택

- 정적 HTML + Vanilla JavaScript
- [Apache ECharts](https://echarts.apache.org) (CDN)
- GitHub Pages 호스팅

## 재현 방법

원본 데이터에서 집계 JSON을 다시 만들려면:

```bash
# 1. 원본 xlsx 파일을 다음 위치에 둔다
#    ~/Downloads/adfile/2019~2025*.xlsx (7개)
#    ~/Downloads/2026년도_정부광고집행내역_*.xlsx (1개)

# 2. 의존성 설치
pip install pandas openpyxl

# 3. 집계 실행 (8개 xlsx 처리에 약 3-5분)
python3 build.py
```

출력:
- `data/overview.json` — 전체 시장 KPI·Top 리스트
- `data/aggregate.json` — `(연도, 기관분류, 기관명, 매체구분, 매체명)` 그룹별 집계 (검색 인덱스)

## 로컬에서 띄우기

```bash
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000
```

`fetch()`가 `file://` 프로토콜에서 막히므로 반드시 HTTP 서버로 열 것.

## 라이선스 / 면책

본 대시보드는 공시 데이터를 기반으로 한 비공식 분석 도구이다. 데이터의 정확성·완전성은 원본 공시에 따른다.
